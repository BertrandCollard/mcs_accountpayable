Inherits from OMCServiceProxy : NSObject

Overview

The main class for the Synchronization Service implementation. Contains methods to make requests and manipulate the cache.

– initialize

Initializes an OMCSynchronization object with the settings from ‘synchronization’ node of OMC.plist * This method OR initializeWithMobileObjectEntities: method, should be the first method called on the OMCSynchronizaiton class
